Build Hexo Blog

Section 1: Preinstallation

  1. Download and install Git, Node.js.

  2. Create Github repository, keep it public and set its name to Username.github.io.

  3. Create an index.html in the repository. Write something and save. Wait a minute and Username.github.io will be available.

  4. Create an empty folder in your disk.

Section 2.2: Connect to github

  1. Edit C:\Users\Username\.gitconfig file. Add this config at the end.

  2. Generate the SSH key by using ssh-keygen -t rsa -C "Your Github Email" in Git Bash. Then press Enter for three times to use default settings (no password). If it successfully completed, it will show an image, which is shown by characters.

  3. Open folder C:\Users\Username\.ssh\, copy the content of id_rsa.pub. Then edit your personal settings in Github. Add an SSH key to connect with Git. Paste the content you copied just now. Save.

  4. Test your SSH connection. Inside the Git Bash, type ssh -T git@github.com. Close the Watt Toolkit or something similar. Expected result is something like Hi username! You've successfully authenticated.... This means you have set correct settings.

Now you have a normal connection with Github (on Git). But you are still not able to upload your blog to your website now. We will make it in the next section.
